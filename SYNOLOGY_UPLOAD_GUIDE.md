## 🚀 업로드 방법

### Safari에서:
1. http://192.168.219.2:5000 접속
2. btf_admin / Ks&js140405 로그인
3. File Station 실행
4. `docker` 폴더로 이동 (없으면 생성)
5. `insightforge-web-deploy.tar.gz` 업로드
6. 우클릭 → 압축 해제
7. Container Manager → 프로젝트 → 생성
8. `/docker/insightforge-web` 선택 → 시작
